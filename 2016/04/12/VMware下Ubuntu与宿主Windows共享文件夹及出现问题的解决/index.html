VMware下Ubuntu与宿主Windows共享文件夹

  • Linux
  • Ubuntu
  • VMware
  • 学习
  • 虚拟机

  1. 在VMware中给虚拟机设置共享目录，鼠标光标对着要共享的虚拟机右键setting->options->shared Folders->Always enabled->confirm

  2. 执行mount命令，之后mnt/hgfs里就可以看到前面设置的共享目录

  1. 出现问题，/mnt/hgfs 下没有共享文件夹
    百度上很多提到的解决办法

验证后，此法无效，这应该是应对Ubuntu 14.04以前版本的。

原因是虚拟机版本和vmwaretools工具的版本不匹配，重新安装tools就可以了。
3.1 先卸载open-vm-dkms和open-vm-tools并且删除配置

3.2 重新安装VMware Tools
3.2.1 虚拟机点击VM->Reinstall vmware tools就弹出有VMware Tools-xxx.tar.gz，把这个文件复制到其他地方
3.2.2解压

3.2.3 得到vmware-tools-distrib，执行该目录下的 vmware-install.pl

最后会出现

安装成功。
现在在/mnt下可以看到hgfs了。

补充：
vmtools设置共享文件夹在linux的mnt/hgfs下却看不到文件夹的问题

这个应该是版本不符合的原因，更新了系统后发现共享文件不见了…
原因是更新了系统后，使用了新的内核引导，导致无法使用VM-tools的共享功能。
解决方法：
在ubuntu下运行虚拟机配置工具重新生成系统initrd文件即可
